HOW TO USE
----------
Streamlabs:        Alert Box -> pick the alert -> Sound -> upload the MP3.
StreamElements:    Overlay -> AlertBox -> the event -> Audio -> upload.
OBS (vanilla):     add a Media Source, or use them as transition stingers.
Volume tip: start the alert sound at ~50% and adjust live — guitars bite.

All sounds are normalized to -14 LUFS so they sit at the same loudness.
