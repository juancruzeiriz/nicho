# Pack 3 (Moto) — fuentes vectoriales

## De dónde sale cada pieza

| Pieza | Origen | Cómo generarla |
|---|---|---|
| Overlay, 3 paneles, 3 pantallas | **Plugin de Figma** | Corré `riffstream-plugin-packs/` con `BUILD = ["moto"]`. Genera los Frames editables; exportás PNG. |
| 8 íconos | **Estos SVG** (`icons/`) | Importalos a Figma (arrastrar el .svg) o usalos directo. Exportá PNG 256×256. |

## Recolorear / exportar
Mismo flujo que el Pack 2 (ver `packs/Pack2_Gym/svg/README.md`). El color de
relleno es `#F4A259`; el detalle interno usa `#1A1A1D` (carbón) para los
recortes. En Figma: importás el SVG y exportás PNG 256×256.
