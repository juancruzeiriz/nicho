# SEO — Keywords y Tags por nicho

> Banco de keywords y etiquetas para todos los packs RiffStream. Basado en el
> research real de eRank/Etsy del README (Semana 1). Etsy permite **13 tags de
> hasta 20 caracteres c/u**. El algoritmo indexa mejor en **inglés**, incluso
> para productos en español — por eso los tags van en inglés y las keywords de
> español se usan en título/descripción (que sí leen los compradores).

---

## 1. Reglas de oro de tags en Etsy

1. **13 tags, siempre los 13.** Cada tag vacío es visibilidad perdida.
2. **Multi-palabra > una palabra.** `gym overlay` rankea mejor que `gym`.
3. **Sin repetir palabras sueltas** entre tags si podés evitarlo (Etsy ya
   combina palabras de título + tags). Mejor cubrir variantes distintas.
4. **≤20 caracteres** por tag (incluyendo espacios). Si se pasa, acortar.
5. **Long-tail para nichos vacíos:** en moto/gym, términos específicos
   (`biker stream pack`) tienen poca competencia y compradores más decididos.
6. Repetir las **keywords del título dentro de la descripción** refuerza el
   match (Etsy pesa la coincidencia título↔descripción).

## 4. Keywords de español para título/descripción

Estas NO van como tags (Etsy prefiere inglés en tags) pero SÍ en el título y
la descripción, porque las leen los compradores hispanohablantes:

- **Genéricas:** overlay para twitch, pack de stream, plantilla obs, pantallas
  de stream, paneles de twitch, alertas de audio, overlay en español.
- **Rock:** overlay guitarra, stream rockero, alertas de guitarra.
- **Gym:** overlay gym, stream fitness, overlay gimnasio, stream de entrenamiento.
- **Moto:** overlay moto, stream motero, overlay biker, stream de rutas.

## 5. Plan de optimización post-lanzamiento

1. Esperá **7-14 días** con el listing activo antes de tocar nada.
2. En **eRank › Listing Audit**, revisá qué keywords traen impresiones reales.
3. Cambiá SOLO los 2-3 tags con peor rendimiento. No reescribas todo de golpe
   (Etsy penaliza temporalmente los cambios masivos de SEO).
4. Si una keyword long-tail (ej. `gym streamer pack`) trae ventas, reforzala
   en el título.
5. Repetí el ciclo cada 2-3 semanas por listing.
